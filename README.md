# My Interactive Resume / Portfolio

A responsive, professional CV website designed using HTML5 and CSS3. It features a clean two-column layout, skill progress bars, and a mobile-friendly design.


## Features
- **Responsive Design:** Looks great on desktop and mobile.
- **Print Friendly:** Optimized for saving as PDF using Ctrl+P.
- **Clean Code:** Structured HTML and CSS for easy editing.

## Technologies
- HTML5
- CSS3 (Flexbox)
- Font Awesome Icons
